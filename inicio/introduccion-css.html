Introducción a CSS

Este es el lenguaje que utilizamos para diseñar una página web.

¿Qué es CSS?

  • CSS significa "Hojas de estilo en cascada"
  • CSS describe cómo se verán los elementos HTML en la pantalla
  • CSS ahorra mucho trabajo, se puede controlar el diseño de múltiples páginas web
  • Las hojas de estilo se pueden reutilizar en varias páginas, creando estilos homogéneos

Ejemplo de CSS

body {
  background-color: lightgray;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}

CSS resolvió un gran problema

HTML nunca fue diseñado para crear páginas web con mucho formato, es decir páginas visualmente atractivas. La complejidad de los sitios web aumentó y los desarrolladores web necesitaban un medio para dar formato a las páginas de manera más precisa. CSS vino a resolver este problema.

Con CSS, los desarrolladores web pueden controlar el estilo y el diseño de múltiples páginas web a la vez. Cualquier cambio de estilo se puede hacer en un solo lugar, en el archivo CSS.
